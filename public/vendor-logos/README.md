# Vendor Logos

Place vendor logo images in this directory with the following filenames:

## Required Logo Files

- `ssf-auto-parts.png` - SSF Auto Parts
- `oreilly-auto-parts.png` - O'Reilly Auto Parts  
- `autozone.png` - AutoZone
- `crow-burlingame.png` - Crow-Burlingame
- `napa-auto-parts.png` - NAPA Auto Parts
- `tri-state-enterprises.png` - Tri-State Enterprises
- `worldpac.png` - WorldPac
- `partstech-catalog.png` - PartsTech Catalog

## Image Specifications

- **Format**: PNG with transparent background (preferred)
- **Size**: 200-400px width recommended
- **Aspect Ratio**: Maintain original vendor logo proportions
- **Quality**: High resolution for sharp display

## Usage

Logos are automatically displayed in the vendor headers on the parts search results page.
